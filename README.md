# US_Airbnb_Market_Trends_2023_Analysis

File: AB_US_2023.csv

## Project Overview

Understand pricing dynamics across different neighborhoods and room types

Identify high-activity hosts and analyze their portfolio characteristics

Compare market segments by city and price tiers

## Dataset Information
Columns: 
- **id**: Unique identifier for each listing
- **name**: 
- **host_id**: Unique identifier for each host
- **host_name**: name of the host
- **neighbourhood_group**: Broad geographic grouping (e.g., "City of Los Angeles", "Unincorporated Areas", "Hollywood")
- **neighbourhood**: Neighborhood name (e.g., "Western Addition")
- **latitude**: latitude coordinate
- **longitude**:  longitude coordinate
- **room_type **: Minimum number of nights required for booking
- **number_of_reviews**: Total count of reviews received
- **last_review**: Date of the most recent review (format: YYYY-MM-DD)
- **reviews_per_month**:Number of listings the host has on Airbnb
- **calculated_host_listings_count**:Number of listings the host has on Airbnb
- **availability_365**: Number of days listing is available in a year
- **city**: City where listing is located

**Link**: https://www.kaggle.com/datasets/kritikseth/us-airbnb-open-data
